The Oracle database has been shutdown and Athena is closing any open connections. This normally occurs around 5am each morning for backups.

If the message has occurred at around 5am, then it can be ignored. Otherwise log a call for the architecture team.

If the message has occurred at around 5am, then it can be ignored. Otherwise log a call for the architecture team.

Thanatos

Error processing ThanatosEntry, written to ErrorSpace.

Some problem has occurred creating the necessary item in Staffware. This could be a problem with the data for Staffware or the Staffware server itself. The item of work has been moved to a holding area awaiting investigation.

Notify the architecture team who should investigate the cause of the problem. The most likely outcome, is that the item will be moved from the holding area to the processing area to be re-procesed.

Thanatos

Removal of entry aborted.

A problem has occurred interacting with the Staffware server. The work item remains in the processing area and will be retried at regular intervals until successfully processed.

Ensure both Staffware and SEO are currently available. Notify the architecture team to investigate the cause of the problem and ensure all outstanding items have been processed.
